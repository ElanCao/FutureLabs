# Data Analysis

Data analysis is the process of inspecting, cleansing, transforming, and modeling data to discover useful information and support decision-making.

## Why This Skill Matters

Data-driven decision making is essential in modern organizations. Analysts bridge raw data and actionable insights.

## Core Techniques

- **Descriptive Statistics**: Summarize data characteristics
- **Data Cleaning**: Handle missing values, outliers, inconsistencies
- **Exploratory Analysis**: Visualize patterns and distributions
- **Hypothesis Testing**: Validate assumptions statistically
- **Regression Analysis**: Model relationships between variables

## Tooling

- Python (pandas, numpy, scipy)
- R
- SQL
- Excel/Google Sheets
- Tableau/Power BI
- Jupyter notebooks

## Assessment Criteria

- Statistical rigor in methodology
- Clear communication of findings
- Appropriate visualization choices
- Reproducible analysis
- Business impact of insights
