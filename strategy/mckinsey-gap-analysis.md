## Gap Identification

### 1. AI-Native Skills Gap

**Missing from SkillTree:**
- AI workflow orchestration
- Human-AI collaboration design
- AI output verification/validation
- Prompt engineering at scale (we have basic `prompt_engineering`)
- Multi-agent system management

**Recommendation:** Create new branch `ai_native` or expand `ai_ml` subdomain:
- `ai_orchestration`
- `human_ai_collaboration`
- `ai_quality_assurance`
- `ai_ethics_oversight`

### 2. McKinsey-Identified "Rising Skills" Gaps

| Rising Skill | Current Coverage | Action |
|--------------|-----------------|--------|
| AI fluency | Partial | Expand definition beyond usage to include management |
| Complex problem-solving | Good | Ensure Level 8-10 reflects human-AI collaboration |
| Teaching/verification | Partial | Add `ai_output_verification`, `ai_training` |
| Judgment/oversight | Minimal | Add `ai_oversight`, `human_in_the_loop` |

### 3. Industry-Specific Gaps

McKinsey notes healthcare, finance, and manufacturing as high-impact sectors. SkillTree coverage:

- **Healthcare:** `clinical_research`, `pharmacology`, `epidemiology` present; add `patient_care_coordination`, `medical_ai_oversight`
- **Finance:** Strong coverage in `finance` branch; add `ai_financial_analysis`, `algorithmic_oversight`
- **Manufacturing:** Limited coverage; consider adding `industrial_automation`, `robotics_coordination`

### Strategic Positioning

SkillTree is uniquely positioned as the **only** skill framework that:
- Maps both human AND AI agent skills
- Covers technical and human-centric skills equally
- Supports 10-level progression for granular capability assessment
- Can model human-AI skill partnerships (human skill + AI skill = team capability)
